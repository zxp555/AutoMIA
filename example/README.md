# Example Inputs

Place the two public example supervision images for AutoMIA in this directory.

The GitHub README assumes that users will run `run.py` with the two images stored under `example/`, for example:

```bash
python run.py \
  --supervision_image1 example/<view1_image> \
  --supervision_image2 example/<view2_image> \
  --train --convert
```

Replace `<view1_image>` and `<view2_image>` with the actual filenames of the two released example images.
